Searchr

Safari extension that provides search of any website using keywords.

How it works

In address bar, type keyword followed by search query. A keyword indicates a web site. For example, y is for youtube, w is for wikipedia. Thus y radiohead looks for radiohead on youtube. You can combine keywords with a + sign. For example, y+w radiohead looks for radiohead on youtube and wikipedia.

For a list of keywords, go to settings (Safari → Preferences → Extensions → Searchr). You can also add new keywords there.
